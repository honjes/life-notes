# Liefe Notes Rewrite

This is a rewrite of the Life Notes app in Vue.js.

## How to run

### Requirements

- Node.js
- [Ionic Cli](https://ionicframework.com/docs/intro/cli)

### Steps

1. Clone the repository
2. Navigate to the `vue-rewrite` folder
3. Run `npm install`
4. Run `ionic serve`
